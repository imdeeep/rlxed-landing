How to install

Three steps to get Rlxed on your Android device.

  1. 01

    Download the APK

    Tap the download button above. Your browser will save the .apk file to your device.

  2. 02

    Allow unknown sources

    Go to Settings → Security → Install unknown apps and allow your browser or file manager to install APKs.

  3. 03

    Open and install

    Open the downloaded file from your notifications or Downloads folder, then tap Install.
